# 📘 Laravel CRUD OOP Project

## 📌 Deskripsi
Project ini merupakan aplikasi **CRUD (Create, Read, Update, Delete)** sederhana menggunakan **Laravel** dengan penerapan konsep **Object-Oriented Programming (OOP)**.

Project ini dibuat untuk memenuhi tugas dan membantu memahami bagaimana konsep OOP digunakan dalam pengembangan aplikasi web.

---

## ⚙️ Fitur
- ➕ Menambahkan data (Create)  
- 📄 Menampilkan data (Read)  
- ✏️ Mengubah data (Update)  
- ❌ Menghapus data (Delete)  

---

## 🧠 Konsep OOP yang Digunakan
- Inheritance  
- Polymorphism  
- Abstract Class  
- Interface  
- Trait  
- Final Class  

---

## 🛠️ Teknologi
- PHP  
- Laravel  
- MySQL  
- Blade Template  

---

## 🚀 Cara Menjalankan Project
```bash
git clone https://github.com/RayyMN/project-crud-laravel-week4.git
cd project-crud-laravel-week4
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
